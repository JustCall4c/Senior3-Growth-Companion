# 🎓 高三成长陪伴助手 | 高考备考成绩管理+心理成长工具
> 专为高三学子与家长打造的「轻量级、本地化、无后台」高考备考数据管理系统，纯前端实现、无隐私泄露风险，适配手机端全尺寸屏幕 

## ✨ 产品核心亮点
### ✅ 【极致轻量化，纯前端无依赖】
- 单文件 `HTML` 编写，**无需部署**，下载即用、打开即用
- 所有数据**永久存储在本地浏览器**，断网可用、无服务器宕机风险，彻底杜绝隐私泄露
- 无第三方埋点、无广告、无数据上报，备考数据只属于你自己

### ✅ 【精准成绩管理，多维度智能校验】
- 支持「语文/数学/英语/物理/化学/政治」6大高考核心科目录入，自动关联各科满分值（语数英150、理化政100）
- 智能总分核对：系统自动计算6科总分，与手动录入总分不一致时弹窗提醒，杜绝录入错误
- 完整记录维度：考试名称、学期、单科分数/排名、总分、班排/校排/级排、考试备注全维度覆盖

### ✅ 【可视化数据复盘，备考趋势一目了然】
- ✅ **单科雷达图**：直观展示单次考试各科「满分率占比」，快速定位强弱项学科
- ✅ **分数趋势图**：折线+面积双图展示「总分/单科」历次考试分数变化，把握成绩波动规律
- ✅ **排名追踪图**：支持班排/校排/级排/单科排名多维度趋势分析，排名变化可视化呈现
- ✅ **极值智能标注**：自动计算展示各科/总分的「历史最高分/最低分」+ 对应考试信息，精准定位进步/退步节点

### ✅ 【专属荣誉勋章，正向激励备考心态】
- 系统根据成绩数据**自动生成荣誉徽章**，包含「单科战神」「逆风翻盘」「稳如泰山」等荣誉标签
- 记录每一次微小进步，用正向反馈缓解备考焦虑，积累备考成就感，适配高三学生心理需求

### ✅ 【双独立「树洞日记」，亲子双向心理陪伴】
> ✨ 核心特色功能，区别于所有同类成绩管理工具
1. **妈妈的树洞**：家长专属私密空间，可记录鼓励话语、育儿心得、焦虑倾诉，带独立密码锁+重置通道，支持日记编辑/删除/修改
2. **yanxi的树洞**：学生专属私密空间，记录备考心情、压力释放、学习感悟，同样带密码保护，打造专属心灵港湾
3. 双树洞数据完全隔离，各自独立加密存储，既满足亲子沟通需求，又保留彼此隐私边界

### ✅ 【考前状态记录，备考全链路追踪】
- 选填「考前睡眠质量、心理压力值、试卷难度」3项心理+状态指标（1-5分评级）
- 完整记录备考全链路数据，不止看分数，更关注「状态-分数」的关联关系，科学调整备考节奏

### ✅ 【AI智能复盘，专业备考指导一键获取】
- 内置「学生版/家长版」双版本AI专属指令，一键复制即可对接 Kimi/豆包/文心一言 等大模型
- ✅ 学生版：温柔导师视角，分析成绩波动、强弱项、学习状态，给出具体提分建议+备考鼓励
- ✅ 家长版：教育心理学视角，安抚家长焦虑情绪，提供「非暴力沟通话术」，教你科学陪伴不添压
- 支持高级模式：对接Gemini API Key，一键调用AI完成深度分析，无需跳转第三方平台

### ✅ 【完善的数据安全体系，备份恢复无忧】
- 全量数据导出：支持「成绩数据」「妈妈树洞」「yanxi树洞」三类数据独立加密导出
- 智能数据恢复：粘贴任意导出的备份文本，系统自动识别格式并完成恢复，兼容格式容错

## 🚀 快速使用指南
### 第一步：使用方式（三选一，任选其一）
1. **直接使用**：将完整HTML代码保存为 `index.html` 文件，用任意浏览器（Chrome/Edge/Safari/微信浏览器）打开即可
2. **本地部署**：无需任何环境，无依赖，纯静态文件，可放在本地文件夹/U盘随时打开
3. **服务器部署**：上传至任意静态服务器/云存储（如GitHub Pages），即可在线访问

### 第二步：核心操作流程
1. **录入成绩**：点击「📝 录入成绩」，填写考试信息、各科分数/排名，系统自动校验总分，完成后保存即可
2. **查看分析**：点击「📊 多维分析」，切换学科标签查看分数/排名趋势，定位学习问题
3. **记录心情**：在「妈妈的树洞/彦熹的树洞」中记录文字，点击锁图标加密，保护隐私
4. **备份数据**：点击「备份/恢复」，选择需要导出的数据类型，完成备份，防止数据丢失
5. **AI分析**：点击「🤖 AI智能复盘」，复制对应指令到AI平台，获取专业备考建议

## 🛡️ 数据安全说明
> ✅ 重中之重，放心使用
1. 所有数据（成绩、日记、密码、配置）均通过 `localStorage` 存储在**本地浏览器**，**不会上传到任何服务器**
2. 树洞日记采用「本地密码校验」，密码存储在本地，无明文传输，无破解风险
3. 提供「重置码」机制，忘记密码可通过专属重置码解锁，数据永不丢失
4. 支持全量数据导出备份，可将数据保存至本地文件，双重保障
